Shebang

Shebang, hashbang, cerquilha-exclamação. Você já deve ter reparado nestes caras em um script qualquer de Shell ou de outra linguagem interpretada.

A sua função é muito simples! Indicar em qual interpretador terá que executar o programa que vem abaixo. No portugol, Hey shell! O interpretador que você deve usar é bash, zsh, ksh e por ai em diante.

Exemplo:

#!/bin/bash   Chama interpretador Bourne again shell
#!/bin/zsh    Chama interpretador Z-shell
#!/bin/ksh    Chama interpretador Korn Shell
#!/bin/fish   Chama interpretador Fishl

Como a Shebang começa com uma cerquilha #, o mesmo não será "interpretado" como um comando pelo shell por se tratar de um comentário! Mas sim passará uma informação por trás dos panos. Se você manja de HTML, é como se fosse alguma informação dentro da tag <head></head>.

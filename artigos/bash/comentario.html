Comentário

Comentários em Shell Script são feitas de uma forma muito simples, bastando adicionar uma cerquilha e logo após o que será comentado! O mesmo não é interpretado pelo bash, então você pode ter 2 linhas de código e 200 comentários que não vai pesar quando executar. Claro o tamanho do script irá aumentar, mas nada além disso. 

# Isto é um comentário no Shell bash.
# Perceba que não será interpretado...
# Uhulll Tô gostando disso! Eu posso escrever um poema?

Mas com grandes poderes vem grandes...??? comentários claro. Existem algumas formas de criar comentários em seus programas:

Em linha

Este tipo de comentário é o que você vai mais usar. 

# Vamos executar o comando ls para listar alguma coisa no diretório home do slackjeff.
ls ~/Music

Meia linha

É um comentário mais direto e muito utilizado em meia linha. 

ls ~/Music # Listando diretório Music em slackjeff.

Bloco

É o mais completo, como o próprio nome já diz. Aqui você pode escrever textão mesmo, tudo que precisa para alertar você no futuro, ou apenas para explicar o que aquele trecho de código faz. 

##############################################################################
# Este trecho é complexo, por isso vou usar o comentário em bloco.
# Bom, vamos listar no diretório Music na home do usuário para saber
# se existe uma música do Metallica por lá.
# Se existir vamos continuar e enviar todas músicas do Metallica para meu
# Servidor com rsync.
##############################################################################
ls ~/Music/*Metallica*
